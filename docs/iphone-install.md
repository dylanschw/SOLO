# Installing SOLO on iPhone SOLO is a Progressive Web App, so it can be installed on an iPhone without the App Store. ## Steps 1. Open Safari on the iPhone 2. Go to the deployed SOLO app URL 3. Sign in 4. Tap the Share button 5. Tap Add to Home Screen 6. Confirm the name 7. Tap Add SOLO will now appear like an app icon on the Home Screen. ## Notes - Use Safari for installation - The app must be served over HTTPS - The app should load once while online before offline behavior is tested - Offline workout set logging works after the app has loaded and the user has an active workout session ## Offline workout logging scope Current offline support: - If a workout session is already started, sets can be queued locally when the browser is offline - Pending sets are stored in localStorage - Pending sets can sync to Supabase when internet returns Future offline support: - Start a workout fully offline - Store full workout day snapshots locally - Resolve conflicts between local and server versions - Show a full offline sync status page
